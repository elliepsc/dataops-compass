# DataOps Documentation Sources

Place your markdown documentation files here, organized by tool:
```
documents/
├── dbt/
│   ├── sources.md
│   ├── models.md
│   ├── tests.md
│   └── ...
├── airflow/
│   ├── dags.md
│   ├── operators.md
│   └── ...
└── great_expectations/
    ├── expectations.md
    ├── checkpoints.md
    └── ...
```

## How to populate

Option 1 — Download from official docs:
- dbt: https://docs.getdbt.com
- Airflow: https://airflow.apache.org/docs/
- Great Expectations: https://docs.greatexpectations.io

Option 2 — Use the scraping notebook in notebooks/.
